输入一个绝对路径。为方便测试，初始值取你的 home 目录。

读 / 写文本文件

宿主 preload 提供的 window.AIBricks.fs.pickFile 能弹原生文件选择器；读到路径后交给 window.nodeLab.fs.readText 读内容。 写入直接落在指定路径，请小心使用。

HTTP 请求

preload 里用 Node 的全局 fetch 发请求，完全不受渲染进程的 CORS 限制——能访问任何主机，包括没开 CORS 的 API。这就是桌面应用相对纯 Web 的核心优势之一。 
（尚未运行）

DNS 解析

node:dns 里 resolve4 / resolve6 不经过浏览器拦截， 能直接查到运营商视角的 A / AAAA 记录。

执行命令

通过 child_process.exec 运行任意系统命令。 ⚠ 这是 Node 应用最危险的能力，生产环境必须做白名单 / 参数校验。 本 demo 只用来演示。
