# DNS Resolver · 域名真实 IP 查找

通过多个公共 DNS 服务器交叉解析域名的真实 IP 地址。

## 功能

- **多 DNS 服务器交叉解析**：同时向 Google (8.8.8.8)、Cloudflare (1.1.1.1)、AliDNS (223.5.5.5)、DNSPod (119.29.29.29) 和系统默认 DNS 发起查询，对比结果。
- **单类型解析**：选择 A / AAAA / CNAME / MX / NS / TXT 中的一种记录类型进行解析。
- **全量解析**：一次性查询域名所有记录类型，获得完整 DNS 概览。
- **唯一 IP 提取**：自动去重，展示域名解析到的所有唯一 IP 地址。
- **耗时统计**：每个 DNS 服务器的响应耗时一目了然。

## 构建

```bash
npm install
npm run build
```

构建产物输出到 `ui/` 目录。
